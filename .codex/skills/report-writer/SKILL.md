# Skill: Report Writer

Use this skill when creating project summaries, market notes, decision memos, or progress reports.

## Language
Reports for the user should be in Ukrainian.

## Structure
Use:
- Executive summary;
- What was checked;
- Findings;
- Risks;
- Recommendations;
- Next actions.

## Rules
- Separate facts from assumptions.
- Do not present unverified numbers as confirmed facts.
- State sample size when discussing restaurant data.
