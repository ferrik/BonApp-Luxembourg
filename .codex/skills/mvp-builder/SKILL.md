# Skill: MVP Builder

Use this skill when building BonApp MVP UI, backend, API, and small product features.

## Core rule
Build the smallest working version.

## Allowed MVP features
- category selection;
- 3 restaurant results;
- restaurant detail page;
- CTA buttons;
- tracking;
- basic admin only if requested.

## Not allowed in MVP
- payments;
- auth;
- courier logic;
- cart;
- menu management;
- complex analytics dashboard.

## UX rule
Do not create a generic long directory as the main flow.
The main flow is decision-first:
Category -> 3 results -> detail -> click.
